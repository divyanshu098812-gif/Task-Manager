# 🚀 Nexus Task Manager

An AI-powered Task Management web application built using **Flask**, **MySQL**, and **Groq AI**. It helps users manage daily tasks efficiently with an intelligent AI assistant, analytics dashboard, calendar view, and notification system.

---

## ✨ Features

- ✅ User Authentication (Login & Signup)
- ✅ Add, Edit, Delete Tasks
- ✅ Mark Tasks as Completed
- ✅ Dashboard with Task Statistics
- ✅ Analytics Dashboard
- ✅ Calendar View
- ✅ Smart Notification System
- ✅ AI Chatbot (Groq Powered)
- ✅ Productivity Insights
- ✅ Daily Planning Assistant
- ✅ Task Prioritization
- ✅ Responsive Dark UI

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
cd YOUR_REPOSITORY
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```env
SECRET_KEY=your_secret_key
FLASK_ENV=development

DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=password
DB_NAME=task_manager_db

GROQ_API_KEY=your_groq_api_key
```

Run the application:

```bash
python app.py
```
